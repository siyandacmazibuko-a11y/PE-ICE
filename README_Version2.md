# PE ICE — Website

This repository contains the static one-page website for PE ICE. The site is configured for GitHub Pages. The domain is peice.co.za (CNAME included).

Files:
- index.html — main site
- style.css — styles
- images/logo.svg — site logo (or replace with images/logo.png if you prefer)
- CNAME — custom domain

Deployment: GitHub Pages is enabled on the main branch root.
